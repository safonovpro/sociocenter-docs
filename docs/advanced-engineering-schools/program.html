Основные элементы интерфейса​

Стартовая страница​

Рисунок № 1 — Список разделов программы развития

На странице доступны следующие элементы интерфейса:

  1. Кнопка «Подписать программу» — по нажатии на кнопку пользователь перейдёт на страницу «Подписание» программы развития. Данная кнопка доступна только пользователям с ролью «Ректор» (более подробно про роли пользователей можно прочитать в разделе «Работа с пользователями» документации).
  2. Кнопка «Предпросмотр в PDF» — по нажатии на кнопку пользователь скачает pdf-документ, который будет подписан по нажатию на кнопку «Подписать программу»» (см. пункт 1 данного списка).
  3. Кнопка «Редактировать» — по нажатии пользователь перейдёт на страницу редактирования текущей введённой информации соответствующего раздела программы развития.
  4. Кнопка «Посмотреть» — по нажатии пользователь перейдёт на страницу просмотра текущей введённой информации соответствующего раздела программы развития.

wysiwyg-редактор​

Нет смысла подробно описывать каждый элемент интерфейса wysiwyg-редактора, т. к. это один из самых привычных для пользователей инструментов с базовыми функциями текстового редактора. На анимации № 1 проиллюстрированы все действия, которые вы можете выполнить используя элементы интерфейса wysiwyg-редактора.

Анимация № 1 — Работа с wysiwyg-редактором

Участие в программах и грантах​

Данный элемент интерфейса доступен в разделе № 1 программы развития — «Текущее состояние и результаты развития университета. Целевая модель и её ключевые характеристики» и является сложным, разберём его подробнее.

Представителям университета с помощью данного блока необходимо добавить программы и гранты, которые тематически совпадают с ключевыми направлениями деятельности передовой инженерной школы. Участие университета в программе/гранте должно попадать в период с 2012 по 2022 года.

Рисунок № 3 — Работа с блоком «Участие в программах и грантах»

В рамках блока «Участие в программах и грантах» пользователю доступны следующие элементы интерфейса:

  1. Кнопка «Добавить» — по нажатии на кнопку добавляется новый элемент содержащий информацию о конкретной программе/гранте (см. рисунок № 3, элементы с названиями «Наименование программы № *»).
  2. Иконка «Стрелка вниз» — по нажатии на иконку пользователю становятся доступны все (элемент интерфейса «аккордеон» разворачивается) поля с данными о конкретной программе/гранте.
  3. Иконка «Крестик» — по нажатии удаляется элемент содержащий информацию о конкретной программе/гранте.
  4. Иконка «Стрелка вверх» — по нажатии на иконку скрываются все (элемент интерфейса «аккордеон» сворачивается) поля с данными о конкретной программе/гранте.
  5. Поле «Наименование программы» — текстовое поле для указания названия программы/гранта.
  6. Поле «Выберите программу» — поля для выбора программы, из конечного списка (доступно значение «Другое»).
  7. Поле «Год начала участия» — целочисленное поле для указания года начала участия в программе/гранте.
  8. Поле «Год окончания участия» — целочисленное поле для указания года окончания участия в программе/гранте, в т.ч. планируемый год. Обратите внимание, что данное поле совместно с полем «Год начала участия» (пункт 7 данного списка) создаёт кратное количество полей для указания годовых бюджетов и загрузки годовых отчётов (пункты 9 и 10 данного списка соответственно).
  9. Поле «20** год (тыс. руб.)» — целочисленное поле для указания бюджета за конкретный год программы/гранта.
  10. Поле «Загрузить годовой отчёт» — элемент интерфейса позволяет пользователю загрузить файл с компьютера. Важно, что в целях сокращения количества вводимой информации университетами, допускается вместо прикрепления годового отчёта к каждому году, загрузить один итоговый отчёт к году окончания программы/гранта или крайнего отчёта (2021 год) по незавершённым программам/грантам.
  11. Поле «Основные задачи участия» — wysiwyg-редактор для ввода данных.
  12. Поле «Достигнутые результаты» — wysiwyg-редактор для ввода данных.

Кнопки​

На каждой странице раздела программы развития в режиме редактирования и просмотра доступны следующие кнопки:

  1. Кнопка «Сохранить» (отсутствует в режиме просмотра) — по нажатии пользователь выполнит сохранение данных соответствующего раздела программы развития.
  2. Кнопка «Назад» — по нажатии пользователь перейдёт к списку разделов программы развития. Обратите внимание! Если кнопка нажата в режиме редактирования, без предварительного сохранения данных введённые/скорректированные данные будут утеряны.

Подписание программы развития​

Рисунок № 6 — Страница подписание документов

На странице доступны следующие элементы интерфейса:

  1. Поле «Выберите сертификат для подписи» — позволяет выбрать один из сертификатов доступных на локальном компьютере пользователя. Подробнее о настройке «КриптоПро ЭЦП Browser plug-in» можно прочитать по ссылке.
  2. Кнопка «Подписать» — по нажатии на кнопку произойдёт подписание документа электронно цифровой подписью выбранной в выпадающем списке «Выберите сертификат для подписи» (см. пункт 1 данного списка).
  3. Кнопка «Просмотреть» — по нажатии на кнопку пользователь скачает pdf-документ, который будет подписан по нажатию на кнопку «Подписать» (см. пункт 2 данного списка).

Отправка заявки​

Рисунок № 7 — Отправка заявки

На странице доступны следующие элементы интерфейса:

  1. Кнопка «Отправить заявку» — по нажатии на кнопку произойдёт отправка заявки на участие в конкурсе.
Обратите внимание

Для успешной отправки заявки от Вашей организации необходимо подписать следующие документы:

  1. Заявление на участие в отборе — ректор университета
  2. Справка на участие в отборе — ректор и главный бухгалтер университета
  3. Программа развития — ректор университета
