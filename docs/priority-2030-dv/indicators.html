Показатели

Все показатели представлены списком и сгруппированы согласно нормативным документам в 3 блока:

  • Результаты предоставления гранта
  • Показатели, необходимые для достижения результатов предоставления гранта
  • Финансовые показатели

Под названием каждого показателя выводится таблица с текущими указанными в системе показателями.

При переходе в раздел «Показатели» пользователю откроется страница (см.рис.1), где будут доступны следующие элементы:

  • Вкладки «Показатели программы» и «Показатели ввода» ①.
  • Кнопка «Просмотреть» ②.

Рисунок № 1 – основная страница показателей

Если пользователь нажмёт на вкладку «Показатели ввода»(см.рис.2), ему откроется страница, на которой собраны показатели для ввода, которые уже непосредственно влияют на показатели программы с соответствующей страницы — «Показатели программы».

Рисунок № 2 – вкладка показателей ввода

При нажатии на кнопку «Заполнить» ① пользователю откроется страница для заполнения показателей.

Для заполнения показателей (см.рис.3) нужно ввести цифры в полях ①. После заполнения показателей пользователь должен нажать кнопку «Сохранить» ②, для сохранения показателей. Пользователь может нажать кнопку «Обновить» ④, для обновления графика расчётных показателей.

Рисунок № 3 – заполнение показателей

После возвращения на вклдаку «Показатели программы» (см.рис.1), пользователь может нажать на кнопку «Посмотреть» ②. Ему откроется страница «Подробнее о показателе» (см.рис.4).

Рисунок № 4 – подробнее о показателе

На этой странице пользователь может отредактировать показатели, нажав на кнопку «Редактирование» ①, посмотреть подробнее о расчёте значений ② и перейти к следующим показателям ③. При нажатии на кнопку «Редактирование» пользователь перейдёт на страницу заполнения показателей, как показано в рисунке № 3, после чего он должен заполнить показатели, следуя тем же указаниям.
