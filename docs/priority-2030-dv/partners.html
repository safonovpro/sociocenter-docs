Партнеры

Перечень партнеров​

Пользователь может добавить организаций-партнёров. Для этого в поле «Добавление организации-партнеров» нажать кнопку «Добавить организацию» ①.

Рисунок № 1 - перечень организаций-партнёров

Добавление партнера​

На открывшейся странице (см.рис.2) пользователь указывает, является ли организация иностранной. Если является, пользователь ставить галочку, если нет, то оставляет поле пустым ①.

Затем пользователь вводит название организации-партнёра ② и нажимает кнопку «Сохранить» ③. Кнопка «Назад» переведёт пользователя на другую страницу ④.

Рисунок № 2 – добавление организации-партнёра

Редактирование​

После того, как пользователь добавил организацию-партнёра и вернулся на страницу перечня организаций-партнёров, ему необходимо заполнить информацию о добавленной ранее организации. Для этого необходимо под названием организации нажать кнопку «Редактировать» (см.рис.1). После этого пользователь перейдёт на страницу редактирования информации об организации-партнёре (см.рис.3).

Рисунок № 3 – редактирование организации-партнёра

На странице редактирования пользователь должен указать тип организации ①. Загрузить согласие организации ② и указать объем финансового обеспечения ③. После заполнения данных пользователь нажимает кнопку «Сохранить» ④, для сохранения данных или «Удалить» ⑥, для удаления данных. Кнопка «Назад» ⑤ вернёт пользователя на предыдущую страницу.
