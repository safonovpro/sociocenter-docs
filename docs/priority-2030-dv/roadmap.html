Планируемая деятельность

На странице «Планируемая деятельность» пользователю будут доступны несколько действий (см.рис. 1).

Пользователь может назначить руководителя стратегического проекта, нажав на кнопку «Назначить» ②, после чего ему откроется окно со списком пользователей, где нужно будет найти нужного пользователя и нажать на кнопку «Выбрать».

Пользователь может отредактировать стратегический проект, нажав на кнопку «Редактировать стратегический проект» ③.

Рисунок № 1 – планируемая деятельность

Редактирование стратегического проекта​

Обратите внимание

При заполнении текста в wysiwyg-редакторе можно использовать следующее форматирование:

  • заголовки с 1 по 4;
  • параграфы;
  • списки (нумерованный, маркированный);
  • форматирование текста (жирный, курсив);
  • загруженные с компьютера картинки.

Всё остальное форматирование текста (цвета, эффеты, размеры) будет удалено.

Подробнее о заполнении смотрите в разделе wysiwyg-редактор

После нажатия на кнопку пользователю откроется страница для редактирования стратегического проекта. Здесь пользователю необходимо заполнить показатели ② - ⑦ (см.рис.2). Текст в заполняемых показателях можно редактировать (wysiwyg-редактор). После заполнения показателей пользователь должен нажать кнопку «Сохранить» ⑧. Для возвращения на основную страницу планируемой деятельности, пользователь должен нажать кнопку «Назад» ①.

Рисунок № 2 – редактирование стратегического проекта

На основной странице планируемой деятельности (см.рис.1) пользователь может перемещаться по вкладкам ④. Во вкладке «Основное» пользователь увидит основную информацию стратегического проекта, которая была ранее заполнена им на странице редактирования стратегического проекта.

Научные проекты​

В данной вкладке пользователь может добавить проект, нажав на кнопку «+ Добавить» ① или же найти нужный проект через поисковую строку ② (см.рис.3).

Рисунок № 3 – Научные проекты

При нажатии на кнопку «+ Добавить» пользователю откроется страница для заполнения информации по научному проекту (см.рис.4).

Рисунок № 4 – создание научного проекта

Здесь пользователь должен заполнить следующие данные:

  • название ②;
  • описание проекта ③;
  • описание результата ④;
  • направления проекта ⑤;
  • даты начала и окончания ⑥;
  • чекбоксы ⑦.

После заполнения всей информации пользователь должен нажать кнопку «Сохранить» ⑧. Для перехода на предыдущую страницу нужно нажать кнопку «Назад» ①.

Если пользоатель хочет отредактировать проект во вкладке «Научные проекты» пользователь должен нажать кнопку «Редактировать» под проектом, который он хочет отредактировать. После чего ему откроется такая же страница, как и для создания проекта (см.рис.5). Там пользователь должен выполнить такие же действия, как и при создании проекта, а затем сохранить изменения ⑧.

Рисунок № 5 – страница редактирования научного проекта

Вернувшись во вкладку «Научные проекты», пользователь может нажать под проектом кнопку «Подробнее», после чего перейдёт на страницу проекта (см.рис.6). Здесь пользователь, так же как и на основной странице планируемой деятельности, может назначить ответственного ② и научного руководителя ③. При нажатии на эти кнопки, пользователю откроется список сотрудников, где после выбора необходимого сотрудника нужно нажать кнопку «Выбрать».

Рисунок № 6 – страница «Подробнее» научного проекта

Пользователю будет доступна кнопка «Редактировать» ④ , при нажатии на которую пользователю откроется страница редактирования научного проекта (см.рис. 5)

Также пользователю будут доступны несколько вкладок для заполнения.

Во вкладке «Основное» пользователю будет показана основная информация о научном проекте, которая была заполнена ранее при создании или редактировании проекта.

Во вкладке «Партнёры» пользователь может указать компанию-партнёра (см.рис. 7), выбрав необходимую компанию из списка ① и нажав на кнопку « Добавить» ②. Пользователь может удалить компанию, нажав на кнопку «Удалить» ③.

Рисунок № 7 – вкладка «Партнёры» научного проекта

Во вкладке «Оборудование и ПО» пользователь должен указать оборудование, от которого зависит ход научного проекта (см.рис. 8).

Для добавления оборудования необходимо нажать кнопку «Добавить» ①. Если оборудование уже было добавлено пользователем, можно нажать кнопку «Редактировать» ② для редактирования информации. Также пользователь может удалить добавленное оборудование, нажав на кнопку «Удалить» ③.

Рисунок № 8 – оборудование и ПО

Страница добавления и редактирования оборудования выглядит одинаково (см.рис. 9).

Здесь пользователь указывает следующую информацию:

  • название оборудования ②;
  • тип оборудования ③;
  • наличие оборудования ④;
  • владельца оборудования (Например, компания-партнёр) ⑤.

После заполнения информации пользователю необходимо нажать кнопку «Сохранить» ⑥. Для возвращения на предыдущую страницу необходимо нажать кнопку «Назад» ①.

Рисунок № 9 – информация об оборудовании

Во вкладке «Проектная команда» пользователь может добавить сотрудников, которые будут работать над проектом (см.рис.10). Для этого надо нажать кнопку «Добавить» ①.

Рисунок № 10 – вкладка «Проектная команда»

После этого пользователю откроется страница (см.рис.11), где он может выбрать тип сотрудника ②, добавить нового пользователя, если его нет в списке ③. После нажатия кнопки «Добавить нового сотрудника» пользователю откроется страница, где нужно заполнить информацию о сотруднике. Затем он может назначить сотруднику роль ④ и через список найти необходимого сотрудника ⑤. После заполнения данных необходимо нажать кнопку «Добавить» ⑥. Кнопка «Назад» ① вернёт пользователя на предыдущую страницу.

Рисунок № 11 – добавление сотрудника в проектную команду

Образовательные проекты​

Вернувшись на основную страницу стратегических проектов, пользователь может перейти во вкладку «Образовательные программы». В данной вкладке необходимо будет нажать кнопку «Добавить», после чего пользователю откроется страница создания/редактирования образовательной программы (см.рис.12).

Рисунок № 12 – создание образовательной программы

На данной странице пользователь должен заполнить поля ② - ⑪ и прикрепить необходимые файлы, после чего необходимо нажать кнопку «Создать» или «Сохранить» ⑫ в зависимости от режима страницы.

Так же как и на вкладке «Научные проекты», пользователь может перейти на подробную страницу проекта, нажав на кнопку «Подробнее».

Подрбная страница образовательного проекта по заполнению мало чем отличается от научного. На данной странице пользователь может назначить ответственного ②, руководителя ③, перейти в режим редактирования ④ образовательоного проекта или перемещаться по вкладкам проекта ⑤ (см.рис.13).

Рисунок № 13 – страница "Подробнее" образовтельного проекта

Вкладки образовательного проекта заполняются так же как и вклдаки научного проекта.

В отличие от научных проектов, во вкладках образовательного проекта есть вкладка «Связь с научным проектом» (см.рис.14). Чтобы добавить связь с научным проектом, необходимо выбрать нужный проект из списка и нажать кнопку «Добавить».

Рисунок № 14 – связь с научным проектом

Вкладка «Кадровое обеспечение» образовательного проекта заполняется так же как вкладка «Проектная команда» научного проекта (см.рис. 10-11).

wysiwyg-редактор​

Нет смысла подробно описывать каждый элемент интерфейса wysiwyg-редактора, т. к. это один из самых привычных для пользователей инструментов с базовыми функциями текстового редактора. На анимации № 1 проиллюстрированы все действия, которые вы можете выполнить используя элементы интерфейса wysiwyg-редактора.
