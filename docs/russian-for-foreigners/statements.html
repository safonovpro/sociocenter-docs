Вкладки​

Для начала работы в системе необходимо после регистрации заполнить заявление на включение в реестр. Также можно изменить данные через вкладку «На изменение» и подать заявку на исключение из реестра при помощи вкладки «На исключение».

  • Вкладка «На включение»
  • Вкладка «На изменение»
  • Вкладка «На исключение»

Для перехода во вкладку необходимо нажать на нужную пользователю вкладку.

Чтобы создать новое заявление на любой из этих вкладок необходимо нажать кнопку создать новое заявление.

Рисунок № 1 — Главная страница «Заявления»

Вкладка «На включение»​

После нажатия на кнопку создать новое заявление пользователю станут доступны три информационных блока, которые необходимо заполнить:

  • Общий раздел
  • Соответствие критериям
  • Места проведения экзаменов

Рисунок № 2 — Страница заявления, вкладка «На включение»

Во вкладке доступны следующие элементы интерфейса:

  1. Кнопка «Редактировать» — при нажатии пользователь перейдёт на страницу редактирования текущей введённой информации по блокам «Общий раздел», «Соответствие критериям», а также «Места проведения экзаменов»
  2. Кнопка «Посмотреть» — при нажатии пользователь перейдёт на страницу просмотра текущей введённой информации по блокам «Общий раздел», «Соответствие критериям», а также «Места проведения экзаменов»
  3. Кнопка «Перейти к подписанию» — при нажатии на кнопку пользователь перейдёт на страницу «Подписание» заявления.

Вкладка «На изменение»​

Вкладка «На изменение» позволяет создать заявление для изменения, в случае если необходимо изменить данные. После нажатия на кнопку создать новое заявление пользователю станут доступны два информационных блока, которые необходимо заполнить:

  • Общий раздел
  • Места проведения экзаменов

Рисунок № 3 — Вкладка «На изменение» заявления

На странице доступны следующие элементы интерфейса:

  1. Кнопка «Редактировать» — при нажатии пользователь перейдёт на страницу редактирования текущей введённой информации по блокам «Общий раздел» и «Места проведения экзаменов».
  2. Кнопка «Посмотреть» — при нажатии пользователь перейдёт на страницу просмотра текущей введённой информации по блокам «Общий раздел» и «Места проведения экзаменов».
  3. Кнопка «Перейти к подписанию» — при нажатии на кнопку пользователь перейдёт на страницу «Подписание» изменённого заявления.

Вкладка «На исключение»​

Данная вкладка позволит пользователю подать заявку на исключение из реестра. После нажатия на кнопку создать новое заявление пользователю станут доступны два информационных блока:

  • Общий раздел
  • Основания исключения

Рисунок № 4 — Вкладка «На исключение»

На странице доступны следующие элементы интерфейса:

  1. Кнопка «Редактировать» — при нажатии пользователь перейдёт на страницу редактирования текущей введённой информации по блокам «Общий раздел» и «Основания исключения».
  2. Кнопка «Посмотреть» — при нажатии пользователь перейдёт на страницу просмотра текущей введённой информации по блокам «Общий раздел» и «Основания исключения».
  3. Кнопка «Перейти к подписанию» — при нажатии на кнопку пользователь перейдёт на страницу «Подписание» заявления на исключение.

«Общий раздел»​

«Общий раздел»: режим редактирования​

На данной странице пользователь сможет загрузить учредительные документы, а также документы, подтверждающие полномочия лица на осуществление действий от имени организации.

Рисунок № 5 — Страница «Общий раздел» в режиме редактирования

На странице доступны следующие элементы интерфейса, которые необходимо заполнить:

  1. Загрузка файла с учредительными документами — данный элемент позволяет загрузить файл с компьютера.
  2. Блок радиокнопок — позволяет выбрать из двух значений «Лицо, действующее от имени организации без доверенности» или «Иное лицо».
  3. Загрузка файла — данный элемент позволяет загрузить документы, подтверждающие полномочия лица на осуществление действий от имени организации.
  4. Кнопка «Сохранить» — при нажатии на кнопку введённые на странице данные сохранятся.
  5. Кнопка «Назад» — при нажатии на кнопку пользователь перейдёт назад в предыдущую вкладку.

«Общий раздел»: режим просмотра​

На странице находятся все данные, которые были загружены в «Общий раздел» в режиме редактирования. Однако в режиме просмотра все элементы, относящиеся к данным, будут не доступны пользователю для редактирования.

Рисунок № 6 — Страница «Общий раздел» в режиме просмотра

На странице доступен один элемент интерфейса:

  1. Кнопка «Назад» — при нажатии на кнопку пользователь перейдёт назад в предыдущую вкладку.

«Соответствие критериям»​

«Соответствие критериям»: режим редактирования​

На данной странице пользователь может заполнить информацию о критериях, которая включает в себя информацию об основных и дополнительных профессиональных программах, а также указать информацию о кадрах организации.

Рисунок № 7 — Страница «Соответствие критериям» в режиме редактирования

На странице доступны следующие элементы интерфейса:

  1. Блок чекбоксов — позволяет проставить галочки напротив основных профессиональных программ.
  2. Блок ввода — позволяет пользователю выбрать «Дополнительные профессиональные программы».
  3. Блок полей ввода — позволяет пользователю ввести информацию о педработниках, такую как ФИО, должность, классификатор специальностей.
  4. Блок чекбоксов — позволяет проставить нужные галочки напротив Направления/Специальности педработника.
  5. Загрузка файла с приказом — данный элемент позволяет загрузить приказ о приёме или трудовую копию с компьютера.
  6. Загрузка файла — данный элемент позволяет загрузить документ с согласием на обработку данных.
  7. Кнопка «Добавить» — при нажатии на кнопку откроется дополнительное поле для введения информации о педработнике.
  8. Чекбокс — позволяет поставить необходимую галочку.
  9. Блок ввода — данный элемент позволяет ввести наименование структурной организации.
  10. Загрузка файла — данный элемент позволяет загрузить с компьютера копию локального акта организации.
  11. Загрузка файла — данный элемент позволяет загрузить с компьютера копию штатного расписания подразделения.
  12. Кнопка «Сохранить» — при нажатии на кнопку введённые на странице данные сохранятся.
  13. Кнопка «Назад» — при нажатии на кнопку пользователь перейдёт назад в предыдущую вкладку.

«Соответствие критериям»: режим просмотра​

На странице находятся все данные, которые были загружены в «Соответствие критериям» в режиме редактирования. Однако в режиме просмотра все элементы, относящиеся к данным, не доступны пользователю для редактирования.

Рисунок № 8 — Страница «Соответствие критериям» в режиме просмотра

На странице доступен один элемент интерфейса:

  1. Кнопка «Назад» — при нажатии на кнопку пользователь перейдёт назад в предыдущую вкладку.

«Места проведения экзаменов»​

«Места проведения экзаменов»: режим редактирования​

На данной странице пользователь может заполнить информацию о местах проведения экзамена, а также прикрепить документы, связанные с местом проведения экзамена.

Рисунок № 9 — Страница «Места проведения экзаменов» в режиме редактирования

На странице доступны следующие элементы интерфейса:

  1. Блок радиокнопок — позволяет выбрать место проведения из двух значений «На территории РФ» или «За пределами РФ».
  2. Чекбокс — позволяет поставить необходимую для подтверждения галочку.
  3. Загрузка файла — элемент позволяет загрузить с компьютера копии документов, подтверждающие право пользования помещением.
  4. Загрузка файла — элемент позволяет загрузить с компьютера документ о праве пользования средствами видео- и адиозаписями для осуществления экзамена.
  5. Загрузка файла — элемент позволяет загрузить с компьютера документ о праве пользования оборудованием для проведения экзамена.
  6. Загрузка файла — элемент позволяет загрузить с компьютера документ о праве пользования оборудованием для хранения результатов экзамена.
  7. Кнопка «Добавить» — при нажатии на кнопку откроется дополнительное окно для добавления места проведения экзамена.
  8. Кнопка «Сохранить» — при нажатии на кнопку данные страницы сохранятся.
  9. Кнопка «Назад» — при нажатии на кнопку пользователь перейдёт назад в предыдущую вкладку.

«Места проведения экзамена»: режим просмотра​

Отличие режима просмотра от режима редактирования заключается в отсутствии возможности редактировать информацию.

Рисунок № 10 — Страница «Места проведения экзаменов» в режиме просмотра

На данной странице доступен один элемент интерфейса:

  1. Кнопка «Назад» — при нажатии на кнопку пользователь перейдёт назад в предыдущую вкладку.

Страница «Основания исключения»​

Страница «Основания исключения» в режиме редактирования​

Данная страница позволит пользователю описать причину выхода из реестра, а также подтвердить данное решение.

Рисунок № 11 — Страница «Основания исключения» в режиме редактирования

На странице доступны следующие элементы интерфейса:

  1. Элемент для заполнения — здесь пользователь может ввести текст для основания исключения.
  2. Чекбокс — позволяет проставить галочку для подтверждения исключения.
  3. Ввод комментария — здесь пользователь может оставить комментарий с указанием причины исключения.
  4. Кнопка «Сохранить» — при нажатии на кнопку введённые на странице данные сохранятся.
  5. Кнопка «Назад» — при нажатии на кнопку пользователь перейдёт назад в предыдущую вкладку.

Страница «Основания исключения» в режиме просмотра​

Отличие режима просмотра от режима редактирования заключается в отсутствии возможности редактировать информацию.

Рисунок № 12 — Страница «Основания исключения» в режиме просмотра

На странице доступен следующий элемент интерфейса:

  1. Кнопка «Назад» — при нажатии на кнопку пользователь перейдёт назад в предыдущую вкладку.

Страница «Подписание документа»​

После загрузки всех необходимых для заявления документов, пользователь может перейти на страницу «Подписание». На данной странице пользователь может выбрать электронно-цифровую подпись, которой будет подписано заявление.

Рисунок № 13 — Страница «Подписание документа»

На странице доступны следующие элементы интерфейса:

  1. Поле «Выберите сертификат для подписи» — позволяет выбрать один из сертификатов доступных на локальном компьютере пользователя.
  2. Кнопка «Подписать» — при нажатии на кнопку произойдёт подписание документа электронно-цифровой подписью выбранной в выпадающем списке «Выберите сертификат для подписи» (см. пункт 1 данного списка).
  3. Кнопка «Просмотреть» — при нажатии на кнопку пользователь скачает pdf-документ, который будет подписан при нажатии на кнопку «Подписать» (см. пункт 2 данного списка).
