Электронная подпись

Для запуска процедуры выдачи электронной подписи необходимо перейти на сайт Удостоверяющего центра.

Вход в личный кабинет УЦ​

Введите номер телефона, который был указан Вами в анкете эксперта. На него придет одноразовый код для входа.

Настройка ПК для выпуска сертификата​

Для работы с выпускаемой электронной подписью необходима предварительная настройка компьютера. Для запуска автоматической настройки нажмите «Настроить компьютер».

Установите необходимые отсутствующие компоненты портала установки для завершения настройки компьютера.

После успешной автоматической установки и настройки отсутствующих компонентов перезагрузите браузер.

Выпуск сертификата​

Проверьте правильность заполненной заявки на выпуск электронной подписи.

Выберите место для хранения закрытого ключа выпускаемой электронной подписи.

После успешного выпуска сертификата подтвердите его получение.

Установка сертификата​

Для установки выпущенного сертификата введите пароль, установленный на шаге «Выпуск сертификата».
