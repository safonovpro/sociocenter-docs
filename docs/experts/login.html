Вход и регистрация

Вход в систему​

Рисунок № 1 — Интерфейсы, связанные со входом в систему

На странице доступны следующие элементы интерфейса:

  1. Поле «Электронная почта» — предназначено для ввода электронной почты пользователя системы.
  2. Поле «Пароль» — предназначено для ввода пароля пользователя системы.
  3. Кнопка «Вход» — при нажатии и условии корректного ввода пары адреса электронной почты и пароля пользователь будет авторизован и переведён на одну из закрытых страниц личного кабинета. В случае ошибок интерфейс подскажет причину, по которой не произошло авторизации.
  4. Кнопка «Войти с помощью ЭП» — нажатие на кнопку переводит интерфейс в режим попытки авторизации с использованием ЭЦП. Корректная работа в этом режиме подразумевает установку дополнительного программного обеспечения на компьютер и браузер пользователя. Следуйте инструкциям системы.
  5. Кнопка «Инструкции» — отправит пользователя к данной документации.
  6. Поле «Электронная почта» — предназначено для ввода электронной почты пользователя системы, на которую будет отправлена информация о том, как сбросить пароль.
  7. Кнопка «Назад» — по нажатии на кнопку интерфейс будет переведёт в режим входа в систему.
  8. Кнопка «Отправить» — инициирует механизм отправки на почту, введённую в поле «Электронная почта» (пункт № 8 данного списка), информации, необходимой и достаточной для сброса пароля пользователя. Обратите внимание! В целях безопасности данных система никак не отреагирует на ввод электронной почты, которая не принадлежит ни одному пользователю системы.
  9. Поле выбора сертификата — в данном поле пользователь может выбрать действующий сертификат ЭЦП.
  10. Кнопка «Назад» — по нажатии на кнопку интерфейс будет переведёт в предшествующий режим.
  11. Кнопка «Далее» — после выбора корректного сертификата ЭЦП (см. пункт 09 данного списка) кнопка становится активной, и пользователь может продолжить работу с системой путём нажатия на данную кнопку, авторизовавшись в системе с использованием выбранного сертификата ЭЦП.

Процесс регистрации​

Для регистрации в личном кабинете эксперта каждому кандидату направляется персональная ссылка, при переходе по которой открывается страница проиллюстрированная на Рисунке № 1.

Начало регистрации​

Рисунок № 1 — Начало регистрации эксперта

На странице доступны следующие элементы интерфейса:

  1. Кнопка «Далее» — по нажатии на кнопку пользователь продолжит регистрацию в системе.

Критерии соответствия​

Далее потенциальному эксперту предстоит выбрать из двух вариантов:

  1. Представитель образовательной организации высшего образования и (или) научной организации
  2. Представитель организации реального сектора экономики, и (или) государственной корпорации, и (или) высокотехнологичной компании

В зависимости от выбранного варианта кандидату будет необходимо выбрать и подтвердить список критериев соответствия.

Рисунок № 2 — Критерии представителя первого типа

Разберём типовые элементы интерфейса, которые позволяют пользователю указать и подтвердить критерии (обратите внимание, что наличие описываемых элементов интерфейса варьируется от критерия к критерию):

  1. Переключатель — по нажатии на один из вариантов система отобразит список критериев, необходимый для заполнения.
  2. Галочка — присутствует напротив каждого критерия и активирует для пользователя возможность указать необходимые данные по критерию и загрузить подтверждающие документы (где это требуется).
  3. Аккордеон — по нажатии отображает или скрывает дополнительные поля конкретного критерия.
  4. Поля для ввода — позволяют пользователю ввести данные по критерию (текст или целые числа).
  5. Поле «Файл» — позволяет пользователю загрузить файл с его персонального компьютера.
  6. Переключатель — позволяет пользователю уточнить информацию о критерии.
  7. Информация о заполненности — информирует пользователя о заполненности данных по критериям.
  8. Кнопка «Перейти далее» — после заполнения всех данных по критериям становится активной и позволяет перейти к следующему этапу (страница с контактной информацией) регистрации эксперта.

Рисунок № 3 — Критерии представителя второго типа

Разберём типовые элементы интерфейса, которые позволяют пользователю указать и подтвердить критерии (обратите внимание, что наличие описываемых элементов интерфейса варьируется от критерия к критерию):

  1. Переключатель — по нажатии на один из вариантов система отобразит список критериев, необходимый для заполнения.
  2. Галочка — присутствует напротив каждого критерия и активирует для пользователя возможность указать необходимые данные по критерию и загрузить подтверждающие документы (где это требуется).
  3. Аккордеон — по нажатии отображает или скрывает дополнительные поля конкретного критерия.
  4. Поля для ввода — позволяют пользователю ввести данные по критерию (текст или целые числа).
  5. Поле «Файл» — позволяет пользователю загрузить файл с его персонального компьютера.
  6. Информация о заполненности — информирует пользователя о заполненности данных по критериям.
  7. Кнопка «Перейти далее» — после заполнения всех данных по критериям становится активной и позволяет перейти к следующему этапу (страница с контактной информацией) регистрации эксперта.

Контактные данные​

Далее потенциальному эксперту будет предложено ввести свои персональные данные, в том числе для последующего оформления трудовых отношений с ФГАНУ «Социоцентр», согласно законодательству Российской Федерации.

Рисунок № 4 — Персональные данные эксперта

На странице доступны следующие элементы интерфейса:

  1. Поле «Фамилия» — текстовое поле для указания фамилии эксперта.
  2. Поле «Имя» — текстовое поле для указания имени эксперта.
  3. Поле «Отчество» — текстовое поле для указания отчества эксперта.
  4. Поле «Дата рождения» — поле для указания даты рождения эксперта.
  5. Поле «Индекс» — целочисленное поле для указания почтового индекса.
  6. Поле «Область» — поле для выбора области (доступен полнотекстовый поиск).
  7. Поле «Населённый пункт» — поле для выбора населённого пункта (доступен полнотекстовый поиск).
  8. Поле «Улица» — поле для выбора улицы (доступен полнотекстовый поиск).
  9. Поле «Дом» — поле для указания номера дома.
  10. Поле «Корпус» — поле для указания корпуса дома (при наличии).
  11. Поле «Квартира» — поле для указания номера квартиры (при наличии).
  12. Поле «Серия паспорта» — целочисленное поле для указания серии паспорта гражданина РФ.
  13. Поле «Номер паспорта» — целочисленное поле для указания номера паспорта гражданина РФ.
  14. Поле «Дата выдачи» — поле для указания даты выдачи паспорта гражданина РФ.
  15. Поле «Код подразделения» — поле для указания кода подразделения, указанного в паспорте гражданина РФ.
  16. Поле «Паспорт выдан» — поле для указания данных, кем выдан паспорт гражданина РФ.
  17. Поле «Копия паспорта» — поле, позволяющее загрузить файл с персонального компьютера пользователя.
  18. Поле «Адрес доставки подписи» — поле для указания почтового адреса доставки электронной цифровой подписи.
  19. Поле «ИНН» — целочисленное поле для указания ИНН эксперта.
  20. Поле «СНИЛС» — поле для указания СНИЛС эксперта.
  21. Поле «Основное место работы» — поле для выбора компании (из резидентов РФ), являющейся основным местом работы эксперта (доступен полнотекстовый поиск по названию и по ИНН организации).
  22. Поле «Место работы по совместительству» — поле для выбора компании (из резидентов РФ), в которой эксперт оформлен по совместительству (доступен полнотекстовый поиск по названию и по ИНН организации).
  23. Поле «Область экспертизы (ГРНТИ)» — поле для выбора до трех рубрик ГРНТИ в качестве областей экспертизы.
  24. Поле «Область экспертизы (классификатор РАН)» — поле для выбора одного из классификаторов Российской академии наук в качестве областей экспертизы.
  25. Поле «Полное наименование банка» — поле для указания полного наименования банка (доступен полнотекстовый поиск), в котором у эксперта открыт счёт, для перечисления заработной платы.
  26. Поле «Номер счёта» — целочисленное поле для указания номера лицевого счёта, принадлежащего эксперту, открытого в банке, указанном в пункте 25 данного списка, для перечисления заработной платы.
  27. Поле «БИК» — целочисленное поле для указания БИК банка, указанного в пункте 25 данного списка (проставляется автоматически после указания названия банка в соответствующем пункте).
  28. Поле «Электронная почта» — поле для указания личного адреса электронной почты эксперта.
  29. Поле «Номер мобильного телефона» — поле для указания личного номера мобильного телефона эксперта.
  30. Галочка «Подтвердить» — необходима для продолжения регистрации, подтверждает, что эксперт даёт согласие на описанное в абзаце выше на странице, перед галочкой.
  31. Поле «Отправить данные» — если все данные заполнены корректно, по нажатии пользователь будет перенаправлен на страницу с подписанием данных своей личной электронной цифровой подписью.
  32. Поле «Назад» — по нажатии пользователь будет перенаправлен на страницу с критериями соответствия.

Подписание​

Прежде чем приступить к проверке указанных данных с последующим подписанием, система предложит выбрать ЭЦП, доступную на компьютере пользователя. Работоспособность подписи можно проверить на данной странице.

Рисунок № 5 - Выбор электронной цифровой подписи

На странице доступны следующие элементы интерфейса:

  1. Поле «Выберите сертификат для подписи» — позволяет выбрать ЭЦП, доступную на локальном компьютере.

После успешного выбора корректно настроенной ЭЦП пользователя перенаправит на следующую страницу.

Рисунок № 6 - Проверка данных и подписание

На странице доступны следующие элементы интерфейса:

  1. Кнопка «Скачать файл» — позволяет скачать и проверить загруженный файл.
  2. Блок «Контактные данные» — позволяет перепроверить указанные ранее персональные данные.
  3. Кнопка «Подписать» — по нажатии произойдёт подписание ЭЦП комплекта документов.
  4. Кнопка «Посмотреть» — по нажатии произойдёт скачивание комплекта документов, которые можно подписать по кнопке «Подписать» из предыдущего пункта.
  5. Кнопка «Перейти далее» — становится доступна после подписания и по нажатии отправляет пользователя на страницу с сообщением о завершении регистрации.
  6. Кнопка «Назад» — по нажатии отправляет пользователя назад на страницу с контактными данными.

Регистрация завершена​

Рисунок № 6 - Регистрация завершена

Подписание договора​

После подтверждения заявки, полученной на указанный адрес элетронной почты, при переходе в систему пользователю предоставится возможность заключить договор с ФГАНУ «Социоцентр», заверив его со своей стороны личной ЭЦП.

Рисунок № 7 — Заключение договора с ФГАНУ «Социоцентр»

На странице доступны следующие элементы интерфейса:

  1. Поле «Выберите сертификат для подписи» — позволяет выбрать ЭЦП, доступную на локальном компьютере.
  2. Кнопка «Подписать» — по нажатии произойдёт подписание ЭЦП договора с ФГАНУ «Социоцентр», и после успешного подписания пользователя перенаправит на главную страницу кабинета эксперта.
  3. Кнопка «Посмотреть» — по нажатии пользователь перейдёт к pdf-версии договора с ФГАНУ «Социоцентр».

После подписания договора​

Если подписание договора прошло успешно, пользователю станет доступен кабинет эксперта (см. Рисунок № 8).
