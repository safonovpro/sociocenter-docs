Процесс экспертизы

Для начала работы над Вашими экспертизами необходимо перейти в раздел «Экспертиза» нажав соответствующий пункт в левом, боковом меню (см. ③ на Рисунке № 1).

Рисунок № 1 — Главная страница кабинета эксперта

На странице доступны следующие элементы интерфейса:

  1. Кнопка «Фамилия И. О.» — по нажатии пользователь перейдёт на страницу для своих персональных настроек.
  2. Кнопка «Выйти» — по нажатии произойдёт выход из системы.
  3. Пункт меню «Экспертиза» — по нажатии пользователь будет направлен к списку для экспертизы.

Список университетов для экспертизы​

Рисунок № 2 — Список университетов для экспертизы

На странице доступны следующие элементы интерфейса:

  1. Кнопка «Оценить» — по нажатии пользователь перейдёт к экспертизе конкретного университета.

Экспертиза университета​

При переходе к экспертизе конкретного университета, пользователь попадает на страницу «Программа» во вкладку «Программа развития» (см. Рисунок № 3).

Программа​

Рисунок № 3 — Программа университета

На странице доступны следующие элементы интерфейса:

  1. Элемент «К списку университетов» — по нажатии пользователь вернётся на страницу со списком университетов.
  2. Основное меню экспертизы — позволяет пользователю навигироваться по следующим разделам экспертизы:
    • Программа
    • Материалы для скачивания
    • Оценка
  3. Меню для навигации по программе развития — элемент интерфейса призван упростить навигацию по «Программа развития» университета, т.к., в большинстве случаев, она будет содержать большое количество материалов.
  4. Вкладка «Программа развития» — содержит текстовые и графические материалы описывающие программу развития.
  5. Вкладка «Об университете» — содержит основные данные об университете (ИНН, названия, учредитель и т.д.).
  6. Вкладка «Цифры и факты» — содержит подробную информацию для экспертизы.
  7. Вкладка «Показатели программы» — содержит данные о показателях заявленных университетом.
  8. Ссылка на сайт университета — по нажатии пользователь перейдёт на сайт изучаемого университета.
  9. Скрол — позволяет пользователю «пролистывать» материалы программы развития.

Об университете​

Рисунок № 4 - Информация об университете

Цифры и факты​

Рисунок № 5 - Цифры и факты об университете

На странице доступны следующие элементы интерфейса:

  1. Элемент «Читать больше» — по нажатии станет доступен для чтения весь текст под заголовком «Информация».
  2. Элемент «Участие в программах» — по нажатию пользователя перенаправит на страницу содержащую информацию об участии университета в программах развития университетов.
  3. Элемент «Научно-исследовательская политика» — по нажатию пользователя перенаправит на страницу содержащую аналитическую информацию о научно-исследовательской политики университета.
  4. Элемент «Образовательная политика» — по нажатию пользователя перенаправит на страницу содержащую аналитическую информацию об образовательной политики университета.
  5. Элемент «Качество системы управления» — по нажатию пользователя перенаправит на страницу содержащую аналитическую информацию о качество системы управления университета.
  6. Элемент «Планируемая деятельность» — по нажатию пользователя перенаправит на страницу содержащую аналитическую информацию о планируемой деятельности.

Показатели программы​

Рисунок № 6 - Показатели программы

На странице доступны следующие элементы интерфейса:

  1. Кнопка «Посмотреть» — по нажатии пользователь перейдёт к графикам показателя и его компонентам.

Материалы для скачивания​

В разделе «Материалы для скачивания» находятся и доступны для скачивания все файлы, которые университет прикрепил к оцениваемой экспертом заявке (в том числе подписанная университетом программа развития в виде pdf-файла).

Рисунок № 7 - Материалы для скачивания

На странице доступны следующие элементы интерфейса:

  1. Ссылка «Скачать программу в pdf» — по нажатии пользователь скачает программу, которая была подписана.
  2. Кнопка «Скачать» — по нажатии пользователь скачает файл, на против которого располагалась кнопка.

Оценка​

Для каждого из оцениваемых университетов эксперту доступен список анкет, которые можно условно разделить на 3 типа:

  • Анкеты с численными оценками (см. пример)
  • Анкеты с качественными оценками (см. пример)
  • Анкеты с аналитической запиской (см. пример)

Рисунок № 8 - Список анкет доступных эксперту

На странице доступны следующие элементы интерфейса:

  1. Кнопка «К подписанию» — по нажатии пользователь будет направлен на страницу для подписания всего комплекта анкет по текущему университету (обратите внимание, система не проверяет полноту заполненности анкет).
  2. Кнопка «Оценить» — по нажатии пользователь перейдёт к заполнению конкретной анкеты.

Подписание анкет​

Рисунок № 9 - Подписание анкет

На странице доступны следующие элементы интерфейса:

  1. Поле «Выберите сертификат для подписи» — позволяет выбрать ЭЦП, доступную на локальном компьютере.
  2. Кнопка «Подписать» — по нажатии произойдёт подписание ЭЦП всего комплекта анкет по университету.
  3. Кнопка «Посмотреть» — по нажатии пользователь скачает файл содержащий все анкеты по университету.

Пример анкеты с численными оценками​

Рисунок № 10 - Численные оценки

На странице доступны следующие элементы интерфейса:

  1. Кнопка «Подтвердить» — по нажатии данные анекты фиксируются.
  2. Индикатор «Итого баллов» — информирует пользователя о текущем количестве баллов в анкете у университета и максимальном возможном количестве баллов по всей анкете.
  3. Индикатор «Критериев заполнено» — информирует пользователя о текущем количестве заполненных критериев в анкете от общего числа критериев присутствующих в анкете.
  4. Навигация по анкете — позволяет пользователю навигироваться по анкете.
  5. Блок для оценки — позволяет пользователю выставить оценку по индикатору.
